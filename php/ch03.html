3.1. 项中获取上一页的URL

常常我们在项目中都会获取上一页的URL，纵所周知 PHP 中的 $_SERVER['HTTP_REFERER'] 可以获取到上一页地址，但是无法获取到 header 跳转的URL 我们点击需要授权的页面后，会跳转到登录页面，然后加上后缀 ret_url 来源地址 


<?php
if(!isset($_GET['ret_url'])){
    if(isset($_SERVER['HTTP_REFERER'])){
        $ret = $_SERVER['HTTP_REFERER'];
    }else{
        $ret = 'index.php'
    }
}else{
    $ret = trim($_GET['ret_url']);
}

$ret = strtolower($ret);
