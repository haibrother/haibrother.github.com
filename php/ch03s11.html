3.11. asort 对数组进行排序(升序，不会改变键)


$fruits  = array( "d"  =>  "lemon" ,  "a"  =>  "orange" ,  "b"  =>  "banana" ,  "c"  =>  "apple" );
 asort ( $fruits );


 echo "<pre>";
print_r($fruits);

--- 输出
Array
(
    [c] => apple
    [b] => banana
    [d] => lemon
    [a] => orange
)
