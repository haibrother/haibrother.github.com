4.2. amqp 扩展安装


   wget http://pecl.php.net/get/amqp-1.0.4.tgz
   tar zxf amqp-1.0.4.tgz 
   /usr/bin/phpize5
   ./configure --with-php-config=/usr/bin/php-config5 --with-amqp;
   make && make install
   
   vim /etc/php5/apache2/php.ini
   extension = "amqp.so"
