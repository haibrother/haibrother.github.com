3.10. rsort 对数组逆向排序(降序，会改变键)


$fruits  = array('c'=>"lemon" ,'a'=>"orange" ,  "banana" ,  "apple" );
 rsort ( $fruits );

 echo "<pre>";
print_r($fruits);

-- 输出
Array
(
    [0] => orange
    [1] => lemon
    [2] => banana
    [3] => apple
)
