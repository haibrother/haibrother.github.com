3.14. krsort 对数组按照键名逆向排序(降序)

   
    $fruits  = array( "d" => "lemon" ,  "a" => "orange" ,  "b" => "banana" ,  "c" => "apple" );
 krsort ( $fruits );


 echo "<pre>";
print_r($fruits);

--- 输出
Array
(
    [d] => lemon
    [c] => apple
    [b] => banana
    [a] => orange
)
