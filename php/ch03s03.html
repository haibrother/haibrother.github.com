3.3. php 文件操作

3.3.1. file_put_contents、file_get_contents

#写入文件
#int file_put_contents  ( string $filename  , mixed  $data  [, int $flags  = 0  [, resource $context  ]] )
#FILE_APPEND   如果文件 filename  已经存在，追加数据而不是覆盖。
#LOCK_EX       在写入时获得一个独占锁。
file_put_contents('data.txt', 'test',FILE_APPEND);

#读取文件
file_get_contents('data.txt');

3.3.2. fopen 系列文件操作

3.3.2.1. 累加写入数据

$fp  =  fopen ( 'data.txt' ,  'a+' );
fwrite ( $fp ,  '1' );
fwrite ( $fp ,  '23' );
fclose ( $fp );
