3.9. sort 对数组排序（升序,会改变键）


$fruits  = array('c'=>"lemon" ,'a'=>"orange" ,  "banana" ,  "apple" );
 sort ( $fruits );

 echo "<pre>";
print_r($fruits);

-- 输出
Array
(
    [0] => apple
    [1] => banana
    [2] => lemon
    [3] => orange
)
