3.13. ksort 对数组按照键名排序(升序)

   
    $fruits  = array( "d" => "lemon" ,  "a" => "orange" ,  "b" => "banana" ,  "c" => "apple" );
 ksort ( $fruits );


 echo "<pre>";
print_r($fruits);

--- 输出
Array
(
    [a] => orange
    [b] => banana
    [c] => apple
    [d] => lemon
)
