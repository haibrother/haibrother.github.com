2.1.1. ls:显示文件名

ls -[参数] 文件/文件夹名
-a 列出目录下的所有文件，包括以 . 开头的隐含文件。
-b 把文件名中不可输出的字符用反斜杠加字符编号(就象在C语言里一样)的形式列出。
-c 输出文件的 i 节点的修改时间，并以此排序。
-d 将目录象文件一样显示，而不是显示其下的文件。
-e 输出时间的全部信息，而不是输出简略信息。
-f -U 对输出的文件不排序。
-g 无用。
-i 输出文件的 i 节点的索引信息。
-k 以 k 字节的形式表示文件的大小。
-l 列出文件的详细信息。
-m 横向输出文件名，并以“，”作分格符。
-n 用数字的 UID,GID 代替名称。
-o 显示文件的除组信息外的详细信息。
-p -F 在每个文件名后附上一个字符以说明该文件的类型，“*”表示可执行的普通
文件；“/”表示目录；“@”表示符号链接；“|”表示FIFOs；“=”表示套
接字(sockets)。
-q 用?代替不可输出的字符。
-r 对目录反向排序。
-s 在每个文件名后输出该文件的大小。
-t 以时间排序。
-u 以文件上次被访问的时间排序。
-x 按列输出，横向排序。
-A 显示除 “.”和“..”外的所有文件。
-B 不输出以 “~”结尾的备份文件。
-C 按列输出，纵向排序。
-G 输出文件的组的信息。
-L 列出链接文件名而不是链接到的文件。
-N 不限制文件长度。
-Q 把输出的文件名用双引号括起来。
-R 列出所有子目录下的文件。
-S 以文件大小排序。
-X 以文件的扩展名(最后一个 . 后的字符)排序。
-1 一行只输出一个文件。

--color=no 不显示彩色文件名
--help 在标准输出上显示帮助信息。
--version 在标准输出上输出版本信息并退出。

2.1.2. cat:显示文本文件内容

显示全部文件.  cat filename

创建文件.  cat filename > file

将几个文件合并为一个文件.  cat file1 file2 > file

2.1.3. rm:删除文件

rm -[参数] 文件/文件夹
rm -f  删除所有文件
rm -l  递归删除文件夹或文件内所有内容
rm -i  提示确认是否删除，输入以y开头的任何字符，都将删除文件或文件夹

2.1.4. linux系统时间和硬件时间

因为linux的系统时间原理和window并不是一样的，window的系统时间一定是与bios的时间一致，而linux 的系统时间是在刚开机的时间读取硬件的时间，之后就独立开来，就这会导致系统时间和硬件时间不一致的情况。

系统时间. date 

显示系统时间
root@laob-VirtualBox:/home/laob# date
2012年 10月 30日 星期二 22:02:10 CST
****************************************************
修改系统时间 格式：date 月日时分年.秒
root@laob-VirtualBox:/home/laob# date 103021582012.12
2012年 10月 30日 星期二 21:58:12 CST
*****************************************************
查看具体时间
root@laob-VirtualBox:/home/laob# date "+%H:%H:%S"
22:22:12

硬件时间. clock

调整时间 hwclock.  -w 系统时间是准确的，把硬件时间设定为系统时间一致 -s 相反
