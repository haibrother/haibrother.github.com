4.3. 输出重定向

覆盖和追加 

        
        >: 覆盖
        >\>:追加
        set -c,禁止覆盖信息到已经存在的文件
        set +c,允许覆盖
        >|
        &>: 合并输出流至同一个数据流
      
        #脚本运行 屏蔽错误信息
        ./test.sh >/dev/null 2>&1
