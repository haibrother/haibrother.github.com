4.7. 从win环境写的bash程序移值到linux/unix服务器执行

从win环境写的bash程序移值到linux/unix服务器执行会出现提示错误提示：syntax error: unexpected end of file

#像这种问题的原因是：dos 格式 文件 传输到 linux/unix 系统时 , 会在每行的结尾多一个
vi   dos.txt         
:set fileformat=unix
:wq

#这样再运行，就不会出现 syntax error: unexpected end of file
