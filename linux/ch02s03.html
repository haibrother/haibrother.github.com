2.3. 修改和设置文件权限

2.3.1. 组

组：权限的集合

2.3.2. chmod

u为属主，g为属组，o为其他，a为所有
chmod [u] w file

2.3.3. chown

chown [属主]:[属组] file

2.3.4. getfacl

    getfacl file

2.3.5. setfacl

    #修改文件权限
    setfacl [-m|-x] u:username:rwx file
    setfacl [-m|-x] g:groupname:rwx file
    #设置目录下文件的默认权限
    setfacl [-m|-x] d:u:username:rwx dir
    setfacl [-m|-x] d:g:groupname:rwx dir
    # -m 添加权限 -x 删除权限
    # u为添加用户权限 g添加属组权限
