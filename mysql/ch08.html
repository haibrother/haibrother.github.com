8.1. replace

mysql中replace函数直接替换mysql数据库中某字段中的特定字符串，不再需要自己写函数去替换，用起来非常的方便。 mysql 替换函数replace()

UPDATE `table_name` SET `field_name` = replace (`field_name`,'from_str','to_str') WHERE `field_name` LIKE '%from_str%' 
#table_name —— 表的名字 
#field_name —— 字段名 
#from_str —— 需要替换的字符串 
#to_str —— 替换成的字符串
