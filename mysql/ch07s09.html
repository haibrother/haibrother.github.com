7.9. concat 连接参数产生的字符串。如有任何一个参数为NULL ，则返回值为 NULL

    

mysql> select concat('laobi','weili');
+-------------------------+
| concat('laobi','weili') |
+-------------------------+
| laobiweili              |
+-------------------------+
1 row in set (0.04 sec)

mysql> select concat(null,'laobi');
+----------------------+
| concat(null,'laobi') |
+----------------------+
| NULL                 |
+----------------------+
1 row in set (0.00 sec)
