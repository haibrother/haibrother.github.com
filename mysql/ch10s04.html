10.4. 使用case when then end 重命名 字段中的值选项



mysql> select id,username,status  from members where id in (78,82);
+----+-----------+---------+
| id | username  | status  |
+----+-----------+---------+
| 78 | 666650139 | Succeed |
| 82 | 666115541 | Failed  |
+----+-----------+---------+
2 rows in set (0.00 sec)

#我们可以通过case when 来重命名该值

mysql> select id,username,`status`,case `status` when 'Succeed' then '成功' when 'Failed' then '失败' end as 中文状态  from members where id in (78,82);
+----+-----------+---------+--------------+
| id | username  | status  | 中文状态     |
+----+-----------+---------+--------------+
| 78 | 666650139 | Succeed | 成功         |
| 82 | 666115541 | Failed  | 失败         |
+----+-----------+---------+--------------+
2 rows in set (0.00 sec)
