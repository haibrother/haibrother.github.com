9.1. is

9.1. is

我们常常会找某数据表中为NULL的数据，有人可能会说，直接 SELECT * FROM TABLE WHERE filed=NULL 不就行了，但实际不能这样子，下面有两种写法：


SELECT * FROM TABLE WHERE filed IS NULL
SELECT * FROM TABLE WHERE filed <=> NULL
或者
SELECT * FROM TABLE WHERE filed IS NOT NULL
# <=>主要是针对NULL而使用的
